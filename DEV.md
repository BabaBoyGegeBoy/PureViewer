# PureViewer 开发文档

## 项目概述
基于 WPF (.NET 8) 的图片大图查看器，用于浏览指定目录下的图片，支持静态图和 GIF 动态播放。

## 功能需求

| 功能 | 说明 |
|------|------|
| 选择文件夹 | 弹出文件夹选择对话框，加载目录下所有支持的图片 |
| 大图查看 | 等比例缩放自适应窗口完整显示 (Stretch=Uniform) |
| 图片导航 | 上一张 / 下一张，循环切换 |
| 全屏 | 进入全屏模式 (无边框最大化) |
| 退出全屏 | 恢复窗口原始状态 |
| 键盘快捷键 | 见下方快捷键表 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| ← / A | 上一张 |
| → / D | 下一张 |
| F11 / F | 全屏切换 |
| Escape | 退出全屏 |

## UI 布局

```
┌──────────────────────────────────────┐
│ [选择文件夹]  共 N 张图片 - 路径      │  ← 顶部：文件夹选择栏
├──────────────────────────────────────┤
│                                      │
│                                      │
│            图片展示区域               │  ← 中部：Image (Uniform)
│                                      │
│                                      │
├──────────────────────────────────────┤
│  [上一张] [下一张] [全屏] [退出全屏]  │  ← 底部：控制按钮栏
└──────────────────────────────────────┘
```

- 整体背景色：`#FF3C3C3C`（深灰）
- DockPanel 布局，三段式

## 技术方案

- **框架**: .NET 8 WPF
- **静态图片**: `BitmapImage` 加载，`BitmapCacheOption.OnLoad` 释放文件锁
- **GIF 动画**: `GifBitmapDecoder` 解码帧 + `DispatcherTimer` 逐帧播放，读取 `/grctlext/Delay` 元数据获取帧延迟
- **文件夹对话框**: `System.Windows.Forms.FolderBrowserDialog`（csproj 启用 `UseWindowsForms`）
- **支持格式**: `.jpg` `.jpeg` `.png` `.bmp` `.gif` `.tiff` `.ico`
