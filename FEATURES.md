# PixViewer 功能介绍

## 功能介绍

### 图片浏览
- 支持多种图片格式：JPG、PNG、BMP、GIF、TIFF、ICO
- 支持 GIF 动画播放，自动解析帧延迟
- 支持从文件夹批量加载图片，支持子文件夹扫描

### 阅读模式
- **网格视图**：缩略图形式展示所有图片，方便快速浏览和选择
- **阅读视图**：大图模式查看，支持自适应等比例缩放
- 自动适应窗口大小，图片完整显示

### 导航功能
- 上一张/下一张切换
- 支持键盘快捷键：方向键、A/D 键
- 支持鼠标滚轮滚动浏览

### 全屏模式
- 全屏查看图片，沉浸式阅读体验
- F11 或 F 键切换全屏
- ESC 键退出全屏

### 搜索与筛选
- 按文件名搜索图片
- 按作者筛选
- 按标签筛选

---

## 阅读体验

### 自适应缩放
- 进入阅读模式时，图片自动等比例缩放到窗口大小
- 保持图片完整性，不裁剪内容
- 窗口大小变化时自动重新适应

### GIF 动画支持
- 自动识别 GIF 格式并播放动画
- 遵循原始帧延迟设置
- 切换图片时自动停止并清理资源

### 平滑导航
- 流畅的页面切换效果
- 图片加载状态反馈
- 支持拖拽浏览

## UI 描述

### 主界面布局
- **左侧边栏**：作者列表、标签列表、搜索框
- **中间区域**：网格视图或阅读视图
- **底部工具栏**：导航按钮、页面信息

### 网格视图
- 卡片式布局展示图片缩略图
- 缩略图统一尺寸，带白色背景
- 悬停高亮效果

### 阅读视图
- 深色背景（#1E1E1E）突出图片内容
- 底部控制栏：上一张/下一张按钮、全屏按钮、退出按钮、页码显示
- 支持鼠标拖拽平移

### 颜色方案
- 主背景：深灰色（#1E1E1E）
- 工具栏：中灰色（#3C3C3C）
- 按钮：悬停高亮效果
- 文字：白色（#FFFFFF）

### 交互设计
- 点击缩略图进入阅读模式
- 点击阅读视图空白区域无操作
- 键盘快捷键支持
- 鼠标滚轮支持（暂时禁用）
