# PureViewer

基于 .NET 8 WPF 的图片大图查看器，专为浏览 Pixiv 下载目录设计。

## 功能特性

- 支持格式：JPG、PNG、BMP、GIF、TIFF、ICO
- GIF 动画自动播放，遵循原始帧延迟
- 按作者筛选（从子文件夹名 `作者名_ID` 自动提取）
- 按标签筛选（解析 info.json 中的 tags 数组）
- 按文件名/目录名搜索
- Ctrl+滚轮缩放，放大后鼠标拖拽平移
- 缩略图网格视图（G 键切换，分批加载防卡顿）
- 全屏模式
- 键盘/滚轮快速翻页

## 支持的图片目录结构

```
根文件夹/
├── 作者A_123456/
│   ├── image1.jpg
│   ├── image2.png
│   └── info.json          ← 可选，含 tags 数组
├── 作者B_789012/
│   └── image3.jpg
└── standalone.jpg          ← 根目录图片归入"（根目录）"
```

`info.json` 格式：
```json
{ "tags": ["风景", "人物"] }
```

## 分支说明

项目包含 4 个分支，代表 4 种 UI 布局方案，核心功能完全相同。

### 共同点

- 相同的图片加载/导航/GIF播放/缩放/拖拽逻辑
- 相同的作者/标签/搜索筛选功能
- 相同的快捷键体系
- 相同的缩略图分批加载机制（每批 50 张，滚动触发）
- 相同的 .gitignore（排除 `.vs/`、`bin/`、`obj/`、`*.user`、`.codebuddy/`）

### 不同点

| 特性 | main | branch-A | branch-B | branch-C |
|------|------|----------|----------|----------|
| **布局方案** | 经典三段式 | 浮动面板 | 极简工具栏 | 全屏沉浸式 |
| **顶栏** | 始终可见，水平布局 | 浮动半透明，鼠标移动显示 | 28px 极简条，始终可见 | 无顶栏，边缘触发顶部 Popup |
| **底栏** | 始终可见，按钮居中 | 浮动半透明，鼠标移动显示 | 可折叠，底部边缘 hover 展开 | 无底栏，边缘触发底部 Popup |
| **作者侧栏** | 左侧 180px，按需显示 | 左侧浮动半透明，Tab 切换 | 左侧 170px，Tab 切换 | 左侧 Popup，Tab/边缘触发 |
| **搜索/标签** | 顶栏内嵌 | 顶栏浮动内嵌 | 顶栏内嵌 | 顶部 Popup |
| **右键菜单** | 无 | 无 | 无 | 有（完整功能入口） |
| **启动引导** | 无 | 无 | 有（WelcomePanel） | 有（WelcomePanel） |
| **主题色** | #3C3C3C | #1E1E1E | #2E2E2E | #1E1E1E |
| **面板交互** | 点击按钮 | 鼠标移动→2秒自动隐藏 | hover 展开/3秒折叠 | 边缘触发+5秒自动关闭 |
| **背景色风格** | 中灰 | 深黑灰 | 暗灰 | 纯黑灰 |

### 分支详情

#### main — 经典三段式

传统桌面应用布局：顶栏（文件夹/搜索/标签）+ 中部（图片/缩略图）+ 底栏（导航按钮）。所有控件始终可见，操作直观。

#### branch-A — 浮动面板

全屏式图片展示，顶栏和底栏浮动半透明覆盖在图片上方。鼠标移动时面板出现，2秒无操作自动隐藏，提供沉浸式阅读体验。

#### branch-B — 极简工具栏

28px 极简顶栏始终可见（文件夹/搜索/标签），底栏默认隐藏、鼠标悬停底部边缘展开（3秒自动折叠动画）。Tab 切换作者侧栏。启动时显示欢迎引导。

#### branch-C — 全屏沉浸式

无传统工具栏，所有面板通过边缘触发（顶部/左侧/底部 50px 触发区）弹出 Popup。右键菜单提供完整功能入口。最极致的沉浸式方案。启动时显示欢迎引导。

## 技术栈

- .NET 8 WPF
- `UseWindowsForms=true`（FolderBrowserDialog）
- `<Using Remove="System.Windows.Forms" />` 避免命名空间冲突
- BitmapImage + BitmapCacheOption.OnLoad（静态图）
- GifBitmapDecoder + DispatcherTimer（GIF 动画）
- ScaleTransform + ScrollViewer（缩放平移）
- ObservableCollection + 分批加载（缩略图性能优化）
